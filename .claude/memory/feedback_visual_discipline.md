Meta-rule for all content production:

**Each section requires a visual.** No section ships as text-only. The visual is not illustration — it is the argument rendered in a different medium. If the section can't be visualized, the argument isn't concrete enough.

**Each visual requires data analysis/context explaining the data.** No visual ships without annotation that explains what the reader is looking at and why the data matters. The visual carries the argument; the annotation connects it to the text's causal chain.

**Why:** Mermaid/Pillow-quality diagrams are insufficient for publication. Visuals must be production-grade — the same standard as the writing. This requires proper tooling: design-grade rendering, multimodal AI for generation, and data visualization libraries that produce publication-ready output.

**How to apply:** Before finalizing any section, ask: what does this look like? Before finalizing any visual, ask: what data does this show and what does the data prove? The visual and the data analysis are inseparable.

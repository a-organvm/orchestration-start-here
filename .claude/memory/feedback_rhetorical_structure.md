Pathos, ethos, logos — all three simultaneously in every paragraph, not one then the other.

Structure must cascade: each pool overflows into the next. No "and then and then and then." Every beat causes the next via BUT or THEREFORE.

Narratological algorithms from ORGAN-I (narratological-algorithmic-lenses) are available and should inform written output:
- **South Park (Parker/Stone):** BUT/THEREFORE connectors, never AND_THEN. Causal chains, not sequences.
- **Larry David:** Parallel apparently-independent storylines that collide through bridge elements. Comedy geometry.
- **Waller-Bridge:** Minimum 3 things going on in every scene/paragraph. Layered obstacles create reality.
- **Kubrick:** Non-submersible units — self-contained blocks where the reader draws connections.
- **Aristotle:** Recognition pleasure — experts see the deeper reference, everyone else appreciates the craft.

**Why:** The user has formalized these as implementable algorithms. They are not metaphors — they are structural rules that apply to all content production including LinkedIn posts, essays, documentation.

**How to apply:** Before writing any content, check: Does each paragraph cause the next (not just follow it)? Are pathos/ethos/logos layered simultaneously? Could each section stand alone? Is there recognition pleasure for the informed reader?
